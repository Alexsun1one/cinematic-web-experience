# Template Kits

These files are runnable scaffolds, not finished client deliverables.

Use them only after a route card and web figure spec exist. Replace sample copy, assets, colors, mechanisms, and labels with the user's source material. Client delivery still must pass:

- Material Transform Gate
- First Viewport Specificity Gate
- Template Contamination Gate
- Proof Density Gate
- Beauty Gate

## Kits

- `before-after-intervention.html`: before/after slider for street, lighting, renovation, service outcome, treatment, or proposal future state.
- `inspectable-product.html`: product image/object inspection with drag rotation, zoom, and source-anchored hotspots.
- `particle-identity.html`: local image upload converted into a canvas particle identity reveal.

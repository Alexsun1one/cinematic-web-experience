# Beauty Gate

This is the final editorial gate. Fail under score 4.

## Evidence Required

```text
beauty evidence:
- page path/url:
- source material inventory:
- material -> mechanism map:
- linked skill outputs:
- desktop screenshot:
- mobile screenshot:
- reduced/no-motion screenshot or proof:
- canvas/3D nonblank proof:
- section source anchors:
- offline truth:
```

## Scoring

Score each 1-5:

- material specificity
- first viewport memorability
- transformation clarity
- typography and spacing
- composition stability
- palette/material restraint
- source-anchor proof density
- mobile composition
- motion meaning
- interaction freedom
- simulation truth if applicable
- non-genericness

Pass requires:

- average >= 4
- material specificity >= 4
- first viewport memorability >= 4
- mobile composition >= 4
- no hard gate failures

## Hard Rejects

- could belong to any AI startup
- source asset not visible or transformed
- starter scaffold patterns survive unchanged
- particles/background unrelated to material
- card wall without visual spine
- text clips or horizontal scroll on mobile
- generated asset used without inspection
- linked skill named without output artifact
- inspectable offer with no rotate/zoom/focus/compare/scrub control
- future-state proposal hides the original source or unlabeled assumptions

## Repair by Failure

- generic -> redo route and hero mechanism
- disconnected -> rebind sections to source anchors
- ugly -> choose stronger material family and reduce colors/objects
- over-animated -> freeze, evaluate static composition, restore only meaningful motion
- weak asset -> generate/edit asset before CSS polish
- mobile bad -> redesign mobile composition, not just shrink desktop

# Effects Catalog

Choose effects by purpose. Do not start from "cool".

## Effect To Purpose Matrix

| Effect | Proves / Expresses | Good For | Stack | Gate |
|---|---|---|---|---|
| photo-to-particles | identity emerging from source | portrait, logo, product silhouette, campaign key visual | Canvas, Three.js points, GPGPU | resolved image recognizable |
| mesh dissolve | transformation, reveal, before/after | product activation, material change, proposal shift | Three.js shader, Codrops-style pattern | not decorative disappearance |
| rotating object | inspection and desire | product, artifact, globe, device | Three.js/R3F/Drei controls | rest state readable |
| exploded view | mechanism clarity | hardware, industrial, furniture, SaaS architecture | R3F planes/meshes, GSAP timeline | labels source-backed |
| before/after wipe | current vs future | renovation, lighting, treatment, service outcome | img-comparison-slider, canvas, CSS clip | original preserved |
| focus dive | detail proof | product details, site zones, dashboard panels | camera move, panzoom, DOM overlay | user can exit/reset |
| command center | control and trust | data, finance, logistics, energy, operations | DOM/SVG/canvas, maps, optional WebGL | no fake metrics |
| map/globe arcs | scale and reach | export, tourism, logistics, networks | three-globe, R3F, SVG maps | arcs/points tied to real data |
| physics assembly | component freedom | product parts, education, playful campaign | Rapier, Matter.js, springs | assembly teaches something |
| scroll camera rail | cinematic journey | proposal, culture, launch, walkthrough | GSAP ScrollTrigger, Theatre.js | chapters inspectable |
| material macro | desire and craft | fashion, cosmetics, food, furniture, manufacturing | Image Gen, video/image plate, shader | material is source-specific |
| paper-stage action | explain abstract relation | consulting, education, culture, process | Image Gen + HTML overlay | action cannot be removed |
| kinetic seal | brand memory | logo, campaign, personal IP | SVG path, particles, Motion/GSAP | mark readable at rest |
| data particles/field | volume, signal, swarm | analytics, science, AI, logistics | Canvas/WebGL instancing | maps to source structure |
| ambient world | emotion and place | hospitality, tourism, culture, exhibition | image/video plate, R3F, parallax | not just atmosphere |

## Effect Bundles

### Premium Product

- hero object rotation
- material macro plate
- focus hotspots
- proof labels
- optional exploded detail

### Future-State Proposal

- original/future compare
- staged reveal
- focus zones
- assumptions labels
- scenario scrub

### Trust / Evidence

- source anchor cards
- audit path
- calm reveal
- proof drill-down
- restrained typography

### Data / Operations

- command center
- scenario toggles
- drill-down map/network
- live state illusion only if data source exists
- static fallback

### Playful Physical

- draggable components
- magnetic snap
- gentle collision
- reset
- celebratory but small reward

## Effect Budget

Default maximum:

- one hero effect
- one primary interaction model
- one scroll or timeline system
- one material/polish system
- small microinteractions for affordance

If more effects are needed, each must map to a different proof job.

## Taste Matching

- serious regulated industry: Trust / Evidence + calm microinteractions
- luxury: Premium Product + liquid luxury motion
- industrial: Mechanism + cutaway + deterministic timing
- playful consumer: Playful Physical + particle identity
- data platform: Data / Operations + command center
- culture/tourism: Ambient World + route atlas + poetic motion

## Anti-Effects

Reject:

- random particles
- scroll hijack with no proof
- endless camera spins
- hover-only details
- fake dashboards
- fake physics
- heavy bloom hiding content
- motion that prevents reading

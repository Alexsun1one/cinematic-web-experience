# Industry Atlas

Use this before assuming a request belongs to one of the examples in `SKILL.md`. The skill is for all industries. Industry only changes proof, risk, and aesthetic register; it does not change the core promise: turn the user's material into an interactive visual prototype.

## Belief Jobs

| Belief Job | Client Needs To Believe | Best Proof Mode |
|---|---|---|
| desire | I want this | inspect, material reveal, worldbuilding |
| feasibility | this can actually be built/delivered | before/after, timeline, simulation |
| trust | this is safe/credible/compliant | evidence theater, source anchors, audit trail |
| control | I can operate/change/configure it | configurator, command center, parameter controls |
| scale | this reaches many places/people | map, globe, network, data field |
| transformation | my current state becomes better | compare, scrub, staged reveal |
| understanding | I finally get how it works | cutaway, layered artifact, paper-stage action |
| identity | this feels like us | particle identity, brand world, signature motion |
| ROI | this is worth buying | outcome model, proof clusters, scenario walkthrough |

## Aesthetic Register By Industry

- clinical: white space, precise typography, soft depth, no spectacle without proof
- industrial: metal/glass/light, cutaways, grids, measurable labels
- luxury: material restraint, macro texture, silence, exquisite spacing
- civic: accessible contrast, clear hierarchy, map/evidence orientation
- cultural: atmosphere, archive respect, route/story rhythm
- technical: legible systems, inspectable UI, exact source labels
- playful: motion, characters, interaction, but still source-bound
- financial/legal: sober polish, proof trails, no casino neon or fake certainty

## Unknown Industry Fallback

If an industry is missing:

1. identify the subject type
2. identify the belief job
3. choose the proof mode
4. choose the lowest freedom level that proves it
5. choose an aesthetic register from the list above
6. run the hard gates

Do not ask the user to classify their industry unless the source materials are impossible to interpret.

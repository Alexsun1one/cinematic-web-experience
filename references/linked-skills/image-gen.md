# Image Gen Contract

Use Image Gen when source material is weak, missing, or needs transformation into a hero plate, texture, object, environment, paper-stage world, or cinematic asset.

## Required Output

```text
image-gen output:
- prompt owner: master-styles | paper-operators | article-visual-grammar | cinematic-web-experience
- source reference:
- intended web use:
- generated path:
- native text status:
- must preserve:
- web transformation:
- QA risk:
```

## Web Usage

- Generate before building if asset determines layout.
- Inspect output before embedding.
- Use generated asset as material for depth, particles, texture, gallery, or scroll chapter.
- Overlay exact text in HTML when text fidelity matters.

## Anti-Misuse

- Do not generate a standalone poster and place it in a generic webpage.
- Do not claim generated logos/maps/products are official unless source verifies them.
- Do not use generated material to erase required source facts.

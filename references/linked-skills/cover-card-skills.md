# Cover / Card Skills Contract

Includes `cover-pop`, `quote-cards`, `rank-cards`, and similar card/cover skills.

## Use When

- The page needs a first-viewport cover asset.
- A chapter needs a strong statement slide.
- A ranking, quote, or claim should become an interactive card wall or pinned moment.
- The generated card becomes a source asset for web spatialization.

## Required Output

```text
cover/card output:
- skill:
- card/cover purpose:
- native text status:
- image path:
- web transformation:
- QA risk:
```

## Anti-Misuse

- Do not paste a card as a static image and call it a website.
- Respect each skill's native text rules.
- Use web motion to reveal, deepen, or contextualize the card.

# Sticker / GIF Skills Contract

Includes `sticker-set` and `gif-stickers`.

## Use When

- Brand tone is playful, social, campaign-like, or expressive.
- A page needs micro-loop characters, sticker reactions, or animated social accents.
- The user explicitly asks for GIF/sticker-like assets inside a web page.

## Required Output

```text
sticker/gif output:
- skill:
- route:
- asset path:
- motion role:
- brand-tone check:
- QA status:
```

## Anti-Misuse

- Do not use stickers on a luxury page unless brand tone supports it.
- Do not cheapen client presentation with random cute loops.
- Do not let GIF assets replace the core material transformation.

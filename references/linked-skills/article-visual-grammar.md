# article-visual-grammar Contract

Use for PDFs, decks, articles, reports, technical explanations, educational content, strategy proposals, product methods, and evidence-heavy pages.

## Required Output

```text
article-visual-grammar output:
- source anchors:
- package rhythm:
- skipped sections and why:
- figure/web spec:
- image-quality harness:
- composition stability preset:
- generated/edited asset path if any:
```

## Web Usage

- Source anchors become page sections and proof nodes.
- Figure specs become web figure specs.
- Composition stability preset becomes hero/spine preset.
- Generated images can become hero plates, scroll chapters, or interactive evidence views.

## Anti-Misuse

- Do not use document content as generic marketing copy.
- Do not invent data or causality.
- Do not build a web section without a source anchor.

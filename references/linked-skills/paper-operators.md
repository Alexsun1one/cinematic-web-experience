# paper-operators Contract

Use when a tactile paper-stage scene or faceless folded-paper operator can physically clarify the idea. This is first-class for consulting, culture, education, systems, textile/craft, emotional/life, and mechanism pages.

## Required Output

```text
paper-operators output:
- source anchor:
- domain and mood:
- core action:
- operator family:
- object acted on:
- relation clarified:
- what breaks if removed:
- metaphor world:
- label plan:

```

## Web Usage

- Paper-stage image becomes a scroll world, not just a background.
- Operator action becomes section choreography.
- Blue carrier/path can become the page visual spine.
- Labels may be converted to precise HTML overlays if model text is imperfect.

## Anti-Misuse

- Do not use the operator for decoration.
- Do not force engineering gates/nodes into art, culture, life, textile, food, or emotional pages.
- Do not crop away the operator action or source labels.

# master-styles Contract

Use when the page needs luxury art direction, domain-appropriate register, palette/light/composition tokens, or Image Gen prompt direction.

## Required Output

```text
master-styles output:
- subject:
- desired feeling:
- structure token:
- palette/light token:
- shared restraint:
- anti-pastiche boundary:
- how tokens map to web:
```

## Web Usage

- Turn tokens into CSS variables, lighting, material, camera, spacing, and hero plate prompts.
- Use one or two token sources only.
- The source material stays hero; style never becomes the subject.

## Anti-Misuse

- Do not write "premium / luxury / cinematic" without concrete tokens.
- Do not copy famous works.
- Do not use maximalist tokens when readability or product inspection is the host constraint.

# Luxury Web Harness

Use this when the user wants the page to feel premium, beautiful, shocking, polished, or client-ready.

## Quality Ladder

### W1: Material Reason

- The page exists because of the user's material.
- First viewport shows the subject/material immediately.
- The strongest effect transforms the material.
- The effect cannot be removed without weakening the page.

### W2: Art Direction

Define concrete visual rules:

```text
luxury harness:
- material family:
- palette discipline:
- typography feeling:
- spatial mode:
- composition engine:
- lighting:
- texture:
- motion signature:
- quiet zones:
- one focal surprise:
- anti-generic list:
```

Reject plans that only say beautiful, modern, premium, cool, or futuristic.

### W3: Web Composition Stability

Choose one preset:

| Preset | Use When | Fixed Layout |
|---|---|---|
| Hero Object Theater | one product/person/place must dominate | full-bleed visual + restrained copy + one action |
| Particle Identity Reveal | photo/logo/object becomes particles | gather/scatter field + sharp resolved subject |
| Rotating World | Earth/map/global story | 3D sphere + orbital labels + proof sections |
| Layered Artifact Cutaway | screenshot/product/document mechanism | stacked planes + scroll reveals |
| Evidence Constellation | PDF/report/deck with proof points | clusters + source anchors + zoom chapters |
| Paper-Stage Scroll World | paper-operators generated scene | carrier path + zoom/pan action stations |
| Cinematic Gallery | multiple photos/assets | spatial wall + focused chapters |
| Atlas Plate Interface | taxonomy/portfolio/data | specimen grid + hover/focus details |

One page may have several sections, but only one preset should own the hero and visual spine.

### W4: Motion Direction

Motion must express cause and transformation:

- gather -> reveal
- orbit -> inspect
- split -> explain
- pass through gate -> qualify
- zoom into evidence -> prove
- light sweep -> focus
- fold/unfold -> memory or document
- rotate -> global or object presence

Avoid motion that only says "look, animation".

### W5: Screenshot Beauty Gate

Before final:

```text
beauty gate:
- source material visible:
- material transformation obvious:
- first viewport memorable:
- one focal module:
- typography clean:
- palette controlled:
- mobile composition intentional:
- no generic SaaS/card-wall look:
- canvas/3D nonblank:
- reduced motion usable:
- score 1-5:
- repair if under 4:
```

Do not call it done under score 4.

## Material Families

Choose one dominant family:

- obsidian glass and fine metal
- editorial paper model
- luminous scientific instrument
- museum archive / vitrines
- cinematic black box theater
- field atlas / specimen plate
- luxury product stage
- celestial / orbital observatory
- soft room / atmospheric memory
- high-fashion editorial grid

Avoid beige-only, purple-gradient-only, dark-blue SaaS sameness, bokeh/orb decorations, and heavy glass cards.

## Typography

- Use stable breakpoint sizes, not viewport-scaled font sizes.
- Never let text touch viewport edges.
- Use short on-screen lines; let visuals carry the spectacle.
- Exact Chinese/brand text should be native HTML overlay unless Image Gen text is intentionally native and verified.
- Hero text should not sit in a card.

## First Viewport Rules

The first viewport must show one of:

- transformed user image
- rotating globe/object
- generated hero plate
- live particle reveal
- spatial gallery/object wall
- paper-stage visual world
- real product/place/person asset

If first viewport could belong to any AI startup, it failed.

## Repair Moves

If it lacks luxury:

```text
Reduce colors to 3-5, increase quiet space, choose one material family, generate a stronger hero plate, and let one focal object dominate.
```

If it lacks connection to material:

```text
Make the material the first viewport spectacle and bind each section to a source anchor or asset fragment.
```

If it looks like a demo:

```text
Add client-specific content, proof, and brand/source material; remove effect labels and tutorial-like sections.
```
